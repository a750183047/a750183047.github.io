HTML学习笔记 表单

表单是HTML中用来提交数据的标签，里面有输入框，单选框，多选矿，字符框等多种控件。form是表单的标签。

  • <input type="text" value="请输入姓名" size="50" readonly disabled> 普通字符输入框，type规定了输入的类型，value则是输入框内的内容，如果这里有内容，则显示在输入框中。size是输入框能够输入的字符长度，readonly，只读，输入框无法输入字符，disable，输入框无法被选中。
  • <input type="password" value="密码" size="50"> type为password，输入类型为密码，则输入的字符会被·替换。
  • <input type="hidden" value="隐藏框" size="50"> type为hidden，输入框隐藏，用来给服务器发送一些数据而不对用户可见。 
  • <input type="radio" name="sex" value="man">
    男
    <input type="radio" name="sex" value="woman">
    女

单选按钮，type为radio，通过name设置相同的值规定为一组，达到单选的效果。

  • <input type="checkbox" name="loves" value="eat">
吃饭
<input type="checkbox" name="loves" value="sleep">
睡觉
<input type="checkbox" name="loves" value="study">
学习

type为checkbox，多选选项，通过name设置相同的值为一组，达到多选的效果。

  • <select name="school">
        <option value="硕士">硕士</option>
        <option value="本科">本科</option>
        <option value="专科">专科</option>
    </select>

select，下拉选择框，通过option设置列表项。select有属性mulitple，多选列表。

  • <textarea rows="10" cols="60">这里是一个大的输入框</textarea> 通过属性rows，和cols设置输入框的行和列。
  • fieldset标签，语义化标签，可以给表单分组，进行语义化。通过legend标签进行标题设置。

按钮

  • <input type="button" value="普通按钮" onClick="alert('这是一个普通按钮')">普通按钮标签，通过type设置为button选择。设置onClick添加点击事件。
  • <input type="submit" value="提交按钮">提交按钮，通过type设置为submit，点击后会将表单的内容提交到form标签下的action属性设置的页面，method属性为提交的方式。
  • <input type="reset" value="重置按钮"> 充值按钮，点击后会将表单内的所有控件的内容重置为初始状态。
  • <input type="image" src="http://file.qingxinchengming.cn/image/blog/deploy.png" width="100" height="40"> 图片按钮，通过加载一张图片来作为按钮，功能和提交按钮相同。
