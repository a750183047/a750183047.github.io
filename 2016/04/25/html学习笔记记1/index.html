HTML学习笔记 标签

为了更好的学习安卓和Java，以及开阔知识面，所以决定学习一些前端和后端的知识，对整个安卓的开发流程都有一个大致的了解，首先决定先学习HTML，毕竟现在HTML5都能开发安卓软件了，必须要跟上时代的潮流，多学点东西以后遇到问题也不至于和前后端的同事交流的时候一脸懵逼。

HTML概述

a.html是什么 ： hypertext markup language 超文本标记语言
超文本：音频，视频，图片称为超文本。.
标记 ：<英文单词或者字母>称为标记. 一个HTML页面都是由各种标记组成。
b.作用：编写HTML页面。
c.HTML语言不是一个编程语言(有编译过程)，而是一个标记语言(没有编译过程).
HTML页面直接由浏览器解析执行。

HTML的网络术语

  • 网页 ：由各种标记组成的一个页面就叫网页 .
  • 主页(首页) : 一个网站的起始页面或者导航页面 .
  • 标记:

    称为开始标记

    称为结束标记. 也叫标签.每个标签都规定好了特殊的含义。
  • 元素：

    内容

    称为元素.
  • 属性： 给每一个标签所做的辅助信息。
  • xhtml： 符合XML语法标准的HTML。
  • dhtml：dynamic,动态的。javascript + css + html 合起来的页面就是一个dhtml
  • http：协议标准。用来规定客户端浏览器和服务端交互时数据的一个格式。SMTP 邮件传输协议,ftp:文件传输协议.

HTML的规范

.HTML是一个弱势语言 .html 不区分大小写
.html页面的后缀名是html或者htm(有一些系统不支持后缀名长度超过3个字符，比如dos系统) .html 的结构
1)声明部分。主要作用是用来告诉浏览器这个页面使用的是那个标准。<!doctype html>是HTML5标准。
2)head部分： 不会显示在页面上。作用是告诉浏览器一些页面的额外信息。
3)body部分：我们缩写的代码必须放在此标签內。

HTML的各种标签

排版标签

HTML中通过各种标签规定样式，每个标签都有私有属性，也有共有属性。

  • p: p代表一个段落，有换行的效果
    • 属性：align:对其方式，取值 ：left，right，center
  • br:br表示换行。没有属性
  • hr:hr表示画一条线。
    • 属性：color 线的颜色，size：线的粗细，width：线的长度，有两种写法，绝对值500，相对值 50%，noshade：不要阴影，align：对齐方式。
  • pre:pre预定义格式标签，告诉浏览器不要忽略空格和空行以及换行。
  • div：div为块级标签。必须单独占据一行，可以把一些标签作为一个块共同处理。
  • span：span为块级标签。不用换行，功能和div相同。

图片标签和超链接

  • img:图片标签，属性：src：图片的路径，相对路径和绝对路径。
    width：图片宽度。
    height：图片高度
    alt:当图片加载失败的时候显示的内容。
    title:共有属性，鼠标悬停时显示的文字。
    border：边框。
    align:表示图片相对内容的位置。
  • a:超链接标签。一共有三种。
    • 链接到其他页面，属性：href ,href="a.html"或者href="http://www.songyan.xyz"。
    • 锚。指给超链接起一个名字，作用是连接到本页或者其他页的特定位置。属性：name。name="top"在另一个超链接中<a href="#top">则跳转到该位置，也可以通过<a href="a.html#top"来跳转到a.html页的top锚的位置。
    • 链接到邮箱。<a href = "mailto:work@songyan.xyz">进入我的邮箱</a>
    • 属性：target，告诉浏览器怎么显示目标页。
      _self:在同一个页面中打开。
      _blank:在新的页面中打开。
      _parent:在父窗口打开。
      _top：在顶级窗口打开。

表格标签

  • table:表格。
    • 属性：
    • border，边框
    • width，宽度
    • height，高度
    • align，对齐方式
    • cellpadding，单元格内容到边的距离
    • cellspacing，单元格和单元格之间的距离
    • bgcolor，背景颜色
    • background，背景图片
    • bordercolorlight，表格亮边的颜色，即表格的上，左边框，单元格的下，右边框的颜色
    • bordercolordark，表格暗边的颜色，即表格的下，右边框，单元格的上，左边框的颜色
    • dir，单元格内容的排列方式，有ltr，从左到右，rtl，从右到左，这是一个公共属性，每个标签都可以使用这个属性。
  • tr:行
    • 属性
    • bgcolor,背景颜色
  • td:单元格
    • 属性
    • align，内容的横向对齐方式
    • Valign，内容的纵向对齐方式，有top，middle，bottom
    • width，宽度，绝对值或相对值
    • height，高度
  • th:加粗，相当于td+b
  • caption:表格的标题
    • 属性
    • align，对齐方式
  • thead,tbody,tfood
    • 当表格非常大时，如果不写，则必须等待表格全部加载完才能显示，但是使用该标签的话，会一边下载一边显示。
    • 如果不写，那么浏览器在加载表格的时候会按照从上到下的顺序加载，如果写了，则顺序任意，浏览器会按照 thead,tbody,tfoot的顺序解析。
  • 单元格合并
    • colspan:横向合并
    • rowspan:纵向合并

    • 属性
    • rows，纵向分布框架
    • cols，横向分布框架
    • 写法，一种是绝对值200,*,一种是相对值20%,*

    • bordercolor，框架的颜色
    • name，框架的名字，用来设置要显示的位置
